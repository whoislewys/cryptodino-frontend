F.A.Q

What is CryptoDino?

CryptoDino in a game designed to bring blockchain to a wider audience while enjoying the same dino game we all love.

How does it work?

Players collect NFT eggs and coins while running. Coins can be used to mint $DINO, a governance token used to run the CryptoDino Protocol. Eggs can be incubated to mint playable Dino skin NFTs.

What is $DINO worth

As of now, CryptoDino is purely a proof of concept and has not been deployed to Near Mainnet, meaning $DINO has no real value. However, we hope to move forward with the protocol, deploy to mainnet, and make $DINO truely fungible.

Whats NEAR?

NEAR is an L1 blockchain that CryptoDino has been built for. Read more about it here!
